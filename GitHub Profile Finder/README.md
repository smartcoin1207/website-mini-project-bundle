# Github Profile Finder

## Description
GitHub Profile Finder will fetch the github details like, Name, Number of Repositories, Bio, Number of Followers, Number of Followings and profile pic.

## Stacks Used
* JavaScript Promises
* API  

## API Used
https://api.github.com/user/user_name
